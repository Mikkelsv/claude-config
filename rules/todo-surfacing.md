# Surface Todo Prompts

When `MEMORY.md` contains a pointer to `todo-prompts.md`, read it at the start of each conversation. If there are unchecked items (`- [ ]`), keep them in mind and surface them at natural moments — e.g., between tasks, when asking what to work on next, or when a todo item is directly relevant to the current discussion.

**How to surface:** Mention the item briefly and naturally, like "By the way, you had a note to ___." Don't interrupt focused work — wait for a pause or transition point.

**Don't:** Read the full list aloud unprompted, nag about the same item repeatedly in one session, or treat the list as a task queue that must be worked through in order.
